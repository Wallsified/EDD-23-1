
# Tarea 1: Estacionamiento

<br>

## Dependencias

Usamos ant y el JDK-17 para ésta tarea. 

<br>

## Comandos de Ejecución

En la carpeta de la práctica abre la terminal y ejecuta los siguientes comandos: 

```
ant compile

ant run
```
<br>
